# @angi-ai/angi

## 0.1.0

### Minor Changes

- e4db2ae: Initial release of the `angi` package providing the following core functionalities:

  - **`useAngiComponent` & `AngiComponent`**: First-class React hooks and components to dynamically render UI from AI responses.
  - **`AngiAgent`**: Server-side adapter for Anthropic's Claude to seamlessly integrate AI capabilities into API routes.
  - **Built-in UI Components**: Ready-to-use components like `AngiChatBubble` and `AngiForm` for rapid development.
  - **Streaming Support**: Real-time component and text streaming directly to the client.
  - **Zero-config Next.js Integration**: Designed to work effortlessly with the Next.js App Router.
